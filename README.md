# 🏥 Clinique Santé+

Application web de gestion de rendez-vous médical développée en PHP & MySQL.

---

## 🚀 Fonctionnalités

- Formulaire de prise de rendez-vous
- Enregistrement sécurisé avec PDO
- Authentification administrateur
- Dashboard de gestion des rendez-vous
- Gestion des sessions

---

## 🛠️ Technologies utilisées

- HTML5
- CSS3
- JavaScript
- PHP (PDO sécurisé)
- MySQL
- Git

---

## ⚙️ Installation

1. Cloner le projet :
   git clone https://github.com/adjidomeyfrancois-byte/clinique-sante.git

2. Créer une base de données "clinique"
3. Importer le fichier SQL
4. Modifier config.php avec vos identifiants
5. Lancer avec XAMPP ou serveur local
